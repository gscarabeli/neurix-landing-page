# 🧠 Neurix — Onde a Ciência Encontra o Jogo

Landing page moderna e inovadora do projeto Neurix, uma plataforma que une a ciência com a gamificação.

## 📋 Sobre o Projeto

Neurix é uma landing page minimalista e futurista desenvolvida com HTML5 e CSS puro, apresentando um design moderno com:
- Paleta de cores vibrante com tonalidades roxas, verdes e âmbar
- Efeitos visuais sofisticados (orbs, noise overlay, gradientes)
- Layout responsivo e otimizado para todos os dispositivos
- Interface intuitiva e profissional

## 🎨 Recursos de Design

- **Design Moderno**: Interface limpa com efeitos visuais modernos
- **Cores Customizáveis**: Sistema de variáveis CSS para fácil personalização
- **Responsividade**: Layout adaptável para mobile, tablet e desktop
- **Tipografia Premium**: Fontes Google Fonts (Syne e DM Sans)
- **Efeitos Visuais**: Orbs com blur, noise overlay, e animações suaves

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Styling avançado com variáveis CSS, backdrop filters, e efeitos visuais
- **Google Fonts**: Tipografia profissional

## 🚀 Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Navegue pela página para explorar o conteúdo
3. Redimensione a janela para testar a responsividade

## 🎯 Características

- Navegação fixa com efeito glassmorphism
- Logo na barra de navegação
- Design responsivo com mobile-first approach
- Overlay de ruído sutil para profundidade visual
- Orbs animadas ao fundo

## 📝 Variáveis CSS Disponíveis

A paleta de cores do projeto é definida através de variáveis CSS:

```css
--p50:    #f3f0ff   /* Roxo claro */
--p100:   #e5deff
--p200:   #c9bbfd
--p400:   #9b7ef8
--p500:   #7c5ce8   /* Roxo principal */
--p600:   #6344d0
--p700:   #4a2eb8
--p800:   #321a96   /* Roxo escuro */
--green:  #6ee7b7   /* Verde vibrante */
--green2: #34d399   /* Verde secundário */
--amber:  #fbbf24   /* Âmbar */
--bg:     #0a0612   /* Fundo principal */
--bg2:    #110b1e   /* Fundo secundário */
--bg3:    #1a1030   /* Fundo terciário */
```

## 🔧 Customização

Para personalizar as cores do projeto, edite as variáveis CSS em `:root` dentro da tag `<style>` no `index.html`.
